- Реализуемо при Budget = 0?

**Конкретный вопрос:**

### 3. Snowballing без публичных источников

Изначально планировался **BFS-алгоритм:**
```
Seed URLs → Extract entities → Generate queries → Find new URLs → Repeat
```

**Проблема:**
Если публичных источников нет, snowballing невозможен.

**Вопрос:**
Можно ли делать snowballing из **приватных источников**?

Например:
```
Email от John@company.com 
  → Extract: "John работает в Company"
  → Query: LinkedIn для John + Company
  → Find profile
  → Extract contacts
```

Это разумный подход? Или альтернатива?
